# FUTURE_CS_02 – Phishing Email Detection and Awareness

## 📌 Project Overview

This repository contains the work completed for **Task 2 – Phishing Email Detection and Awareness** as part of the **Future Interns Cyber Security Internship**.

The objective of this project is to analyze email samples, identify phishing indicators, verify sender authenticity, inspect email headers, and classify emails based on their security risk. The project also promotes cybersecurity awareness by educating users on how to recognize and avoid phishing attacks.

---

## 🎯 Objectives

* Analyze email headers to verify sender authenticity.
* Identify phishing indicators in suspicious emails.
* Validate SPF, DKIM, and DMARC authentication.
* Classify emails as Safe, Suspicious, or Phishing.
* Improve cybersecurity awareness through practical analysis.
